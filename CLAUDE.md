# CLAUDE.md - Tech News Aggregator

## Project Overview

A Next.js application that displays the latest tech news across different categories. News content and excerpts are generated by Claude using the Claude Agent SDK, with daily caching for performance.

## Tech Stack

- Bun (runtime & package manager)
- Next.js 14+ (App Router)
- TypeScript
- TailwindCSS
- shadcn/ui components
- Claude Agent SDK (@anthropic-ai/claude-agent-sdk)

## Features

- Multiple tech news categories (AI/ML, Web Dev, Mobile, Cloud, Security, etc.)
- AI-generated news summaries and excerpts via Claude
- Daily cached responses for optimal performance
- Responsive design with shadcn components

## Conventions

- Use camelCase for variables and functions
- Use PascalCase for types, interfaces, and components
- Use kebab-case for file names
- Prefer named exports over default exports
- Use `async/await` for asynchronous code
- Server Components by default, `'use client'` only when needed

## Authentication

The project uses the Claude Agent SDK which supports two authentication methods:

1. **Claude Code Login (Recommended)**: Run `claude` and login to use your Claude subscription
2. **API Key (Fallback)**: Set `ANTHROPIC_API_KEY` environment variable

```
ANTHROPIC_API_KEY=          # Optional: Claude API key (fallback if not logged in via Claude Code)
```

## Caching Strategy

- News data is cached for 24 hours using Next.js caching
